### 🔸 SmartEvent Platform — Microservices Architecture
> A full-featured **Event Management & Ticketing System** built with Java 21 and Spring Boot 3.x using a clean microservices architecture.

| Service | Technology | Responsibility |
|---|---|---|
| `config-server` | Spring Cloud Config | Centralized configuration |
| `discovery-server` | Netflix Eureka | Service Registry & Discovery |
| `api-gateway` | Spring Cloud Gateway | Routing, Auth Filter |
| `user-service` | Spring Boot + Spring Security + JWT | Authentication & User Management |
| `event-service` | Spring Boot + JPA | Event CRUD & State Machine |
| `booking-service` | Spring Boot + OpenFeign | Booking & Seat Management |
| `payment-service` | Spring Boot | Payment Processing |
| `notification-service` | Spring Boot + Kafka | Async Notifications |

**Tech:** `Java 21` · `Spring Boot 3.5` · `PostgreSQL` · `Docker` · `Kafka` · `JWT` · `OpenFeign`
